# Формат входных и выходных данных
 На вход программе подаются следующие параметры:
 1. Путь к файлу, который необходимо проверить на наличие плагиата ( язык - C++ ). 
 2. Путь к директории, содержащей один или несколько файлов, являющихся оригинальными ( предположительно источниками, с которых было "списано" тестируемое решение ). 
 Формат запроса:
```sh
./AntiPlag main.cpp C:\Projects\Student_code
```
 В ходе выполнения программы формируется текстовый файл, в который записывается результат: содержит тестируемый объект фрагменты "чужого" кода или нет.
Формат вывода в файл:
> Testing file: main.cpp 
Result: most likely plafiarism of file: main_by_Vova.cpp
